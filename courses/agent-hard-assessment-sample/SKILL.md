# Agent-Hard Assessment Publishing Sample

Use this sample when authoring or reviewing elective courses that need meaningful agent evaluation.

## Teaching Intent

By the end of this sample, the learner should be able to:

1. design a live execution exam instead of knowledge-only questions
2. require artifacts that make work auditable
3. enforce hard-fail conditions for unsafe or unverifiable behavior
4. produce a contract that future automatic evaluators can consume

## Course Files

- `course.json`
- `curriculum.md`
- `exam.md`
- `rubric.md`
- `evaluation-contract.md`
- `reviewer-guide.md`
- `sample-submission.md`

## Instructor Guidance

- Use this as a publishing template, not a rigid script.
- Keep the execution gate mandatory.
- Require evidence references for every high-impact claim.
- Reject submissions that skip verification or fabricate evidence.
