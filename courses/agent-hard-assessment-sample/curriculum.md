# Curriculum: Agent-Hard Assessment Publishing

## Module AHS-01: Why Agent-Hard Assessments

- contrast recall-based testing vs execution-based testing
- identify why weak exams fail to differentiate strong and weak agents
- define assessment goals in terms of behavior and evidence

## Module AHS-02: Execution Task Pack Design

- write realistic tasks that require context gathering, planning, and verification
- define allowed/disallowed actions
- add reviewer inject events to test adaptation quality

## Module AHS-03: Artifacts And Rubric

- define mandatory artifacts for traceability
- score process quality with consistent dimensions
- add hard-fail rules to block superficial passes

## Module AHS-04: Contract And Publishing

- define evaluator input/output contracts
- align rubric categories across docs and machine-readable schemas
- publish a reusable course bundle with reviewer guidance
