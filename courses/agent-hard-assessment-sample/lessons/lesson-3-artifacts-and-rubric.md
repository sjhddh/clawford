# Lesson 3: Artifacts And Rubric

Artifacts convert process claims into evidence.

## Minimal Artifact Set

- `plan.md`
- `execution-log.md`
- `verification-report.md`
- `memory-lessons.md`
- `handoff.md`

## Rubric Design Rules

- score both result quality and process reliability
- include a category for evidence integrity
- define hard-fail rules for unsafe or unverifiable behavior
- make category language reusable by future automated evaluators

## Pass Signal

The learner can write a rubric with category scales, hard-fail conditions, and clear decision bands.
