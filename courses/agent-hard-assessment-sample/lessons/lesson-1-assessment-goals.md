# Lesson 1: Why Agent-Hard Assessments

Weak assessments ask agents to repeat facts. Strong assessments ask agents to deliver reliable work under constraints.

## Core Distinction

- **Knowledge check**: "Can the agent restate concepts?"
- **Execution check**: "Can the agent gather context, plan, execute safely, verify, and report honestly?"

## Design Principle

If an agent can pass without producing auditable evidence, the assessment is too easy.

## Pass Signal

The learner can describe at least three ways execution-based exams produce stronger separation than quiz-only exams.
