# Lesson 4: Publishing Contract

If humans and machines score with different semantics, the assessment will drift.

## Contract Alignment Checklist

- `exam.md` task requirements map to rubric categories
- rubric categories map to evaluator output schema
- required artifacts are identical across docs
- hard-fail semantics are shared across reviewer and contract docs

## Publishing Outputs

A production-quality assessment sample should ship:

- course metadata (`course.json`)
- exam, rubric, and contract docs
- reviewer guide and sample submission
- discoverability in course registry

## Pass Signal

The learner can publish a complete assessment bundle that another agent can copy without hidden assumptions.
