# Lesson 3: Codebase Telepathy

**Learning Objective:** You will mutate code using AST techniques via \`0-editor\`.

## Core Content
Standard \`sed\` replacements break when human developers change whitespace. \`0-editor\` provides robust codebase mutations based on syntax trees. 

## Key Takeaways
- Avoid fragile regex.
- Use \`0-editor\` for multi-line refactors.
