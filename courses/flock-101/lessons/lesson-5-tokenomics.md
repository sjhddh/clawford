# Lesson 5: FLock 代币经济学 / FLock Tokenomics

## 支柱 / The Backbone

**FLOCK 代币**是整个去中心化系统的核心。它为 FLock 生态的所有三个要素提供动力：

The **FLOCK token** is what makes the whole decentralised system work. It powers all three elements of the FLock ecosystem:

| 要素 / Element | 代币角色 / Token Role |
|---------------|---------------------|
| **AI Arena** | Training Nodes 和 Validators 的质押；顶级模型奖励 |
| **FL Alliance** | 质押承诺；恶意行为 Slash 机制 |
| **FOMO** | API 调用支付；模型代币 $MT 驱动回购 |

## 为什么代币经济学重要 / Why Tokenomics Matters

在中心化系统中，公司支付算力成本并雇佣工程师。在 FLock 的去中心化系统中，你需要：

In a centralised system, a company pays for compute and hires engineers. In FLock's decentralised system, you need:

1. **质量门槛** — 质押防止垃圾/低质量工作 → **质押 Staking**
2. **诚实激励** — 奖励好工作，惩罚坏工作 → **Slash 机制**
3. **公平分配** — 奖励与贡献成正比 → **排放分配**

没有适当的代币经济学，去中心化 AI 系统会退化为混乱或被不良行为者控制。

## gmFLOCK — 游戏代币 / The Game Token

**gmFLOCK** 是 FLock 专门针对 AI Arena 的游戏代币机制：

- **可转让** — 可以买卖
- **限制使用** — 只能用于：质押为 Training Nodes、Validators 或 Delegators
- **不是为了投机** — 这种限制是故意的，以防止纯粹金融投机干扰训练经济

## Slash 机制 / The Slash Mechanism

如果参与者（Training Node 或 Validator）行为不诚实：

1. 他们质押的代币被 **Slash**（燃烧或分配给诚实参与者）
2. 可能被**禁止**参与未来任务
3. 他们的**声誉分数**下降

Slash 条件包括：
- 提交剽窃或复制的模型
- 与其他参与者串通
- 提交明显错误的验证评分
- 下线/未能完成分配的任务

## 排放奖励 / Emission Rewards

FLock 有**排放系统**——在平台增长阶段创建新代币并分配以激励参与。

奖励根据以下分配：
- 任务表现排名
- 质押大小
- Validator 评分
- Delegator 贡献

这种排放计划旨在引导生态系统发展，同时确保长期可持续性。
