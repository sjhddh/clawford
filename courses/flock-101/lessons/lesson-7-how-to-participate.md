# Lesson 7: 如何参与 / How to Participate

## 根据你的背景选择路径 / Paths Based on Your Background

### 对于开发者 / 数据科学家 / For Developers / Data Scientists

你可以作为 **Training Node** 参与：
You can participate as a **Training Node**:

1. 访问 [train.flock.io](https://train.flock.io)
2. 质押 FLOCK 代币 / Stake FLOCK tokens
3. 选择一个活跃任务 / Choose an active task
4. 提交你训练的模型 / Submit your trained model
5. 如果你的模型排名好，获得奖励 / Earn rewards if your model ranks well

**前置要求 / Prerequisites:** ML 训练技能、GPU 算力（可以从云服务商租用）

### 对于 ML 验证者 / For ML Validators

你可以作为 **Validator** 参与：
You can participate as a **Validator**:

1. 质押 FLOCK 代币 / Stake FLOCK tokens
2. 获得分配的验证任务 / Get assigned validation tasks
3. 诚实评分提交的模型 / Score submitted models honestly
4. 获得验证奖励 / Earn validation rewards

**前置要求 / Prerequisites:** 理解模型评估指标、有质押代币

### 对于投资者 / 代币持有者 / For Investors / Token Holders

你可以作为 **Delegator** 参与：
You can participate as a **Delegator**:

1. 将 FLOCK 委托给可信的 Training Node 或 Validator
2. 分享他们的奖励 / Earn a share of their rewards
3. 根据业绩记录和奖励分享比例选择委托对象

**前置要求 / Prerequisites:** FLOCK 代币、研究委托对象的表现

### 对于企业 / 组织 / For Businesses / Organisations

你可以通过以下方式参与：
You can participate through:

- **FL Alliance** — 贡献私有数据以改进 AI 模型
- **FOMO** — 通过发行模型代币为你的 AI 模型筹资并变现
- **任务创建**（即将推出）— 为社区定义 AI 任务

## 2026 年路线图 / The 2026 Roadmap

FLock 有活跃的开发路线图：
FLock has an active development roadmap:

- 向社区开放任务创建 / Opening task creation to the community
- 更多模型类型和任务类别 / More model types and task categories
- 将 FL Alliance 扩展到更多行业 / Expanding FL Alliance to more industries
- 改进委托 UI 和工具 / Improved delegation UI and tooling
- 跨链扩展 / Cross-chain expansion

## 你的个人参与计划 / Your Personal Participation Plan

**练习 / Exercise:** 根据你的背景回答以下问题：

1. 你的主要技能是什么？（编程/数据科学/投资/商业）
2. 你有 GPU 算力吗？（有/没有/只有云端）
3. 你想投入多少资本？（用于质押/委托）
4. 你对哪个领域感兴趣？（医疗/金融/通用 AI）

**示例答案 / Example answers:**
- 开发者 + GPU → 成为 Training Node
- 数据科学家 + 无 GPU → 成为 Validator 或 Delegator
- 投资者 + 低技术能力 → Delegator
- 拥有私有数据的企业 → FL Alliance 参与者
