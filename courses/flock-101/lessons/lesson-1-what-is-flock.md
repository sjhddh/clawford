# Lesson 1: 什么是 FLock.io？ / What is FLock.io?

## 简短版 / The Short Version

**FLock.io** 是一个去中心化 AI 训练平台 / is a decentralised AI training platform. 

过去，AI 训练数据都被一家公司控制。FLock 让任何人都能贡献算力、数据或专业知识——并获得回报。

In the past, AI training data was controlled by one company. FLock lets anyone contribute computing power, data, or expertise — and get rewarded for it.

**简单类比：** 就像 Kaggle 竞赛 + 区块链 + 联邦学习的结合，但有真实资金激励，没有单一控制点。

**Analogy:** Think of it like Kaggle meets blockchain meets federated learning, but with real money incentives and no single point of control.

---

## 什么是联邦学习？ / What is Federated Learning?

联邦学习（Federated Learning, FL）是一种**不需要任何人聚集原始数据**就能训练 AI 模型的方法。

Federated Learning (FL) is a way to train AI models **without anyone pooling raw data**.

### 传统 AI 训练 / Traditional AI Training:
1. 把所有数据收集到一个地方（你的服务器）/ Collect all data in one place (your servers)
2. 训练模型 / Train the model
3. 完成 — 但现在你拥有了所有人的数据 / Done — but now you have everyone's data

### 联邦学习 / Federated Learning:
1. 数据留在原地（在用户设备或机构服务器）/ Keep data where it is (on user's device or org's server)
2. 本地训练模型 / Train model locally
3. **只分享模型更新（而不是原始数据）** / Only share model updates (not raw data)
4. 合并更新，改进全局模型 / Combine updates to improve the global model

FLock 在此基础上添加了**区块链**，实现：
FLock takes this further by adding **blockchain** to:
- 验证参与者是否诚实 / Verify that participants are being honest
- 自动分配奖励 / Automatically distribute rewards
- 防止作弊或操纵 / Prevent cheating or manipulation

---

## FLock 的核心使命 / FLock's Core Mission

FLock.io 有三大目标 / FLock.io has three goals:

1. **民主化 AI 训练 / Democratise AI training** — 任何人都能贡献数据、算力或模型
2. **保护数据隐私 / Protect data privacy** — 原始数据永不离开源头
3. **对齐 AI 与公众价值观 / Align AI with public values** — 社区决定什么是"好"的 AI

---

## 三大核心产品 / The Three Core Products

FLock 生态有三个相互关联的产品 / FLock's ecosystem has three interlocking products:

| 产品 / Product | 功能 / What it does |
|---------------|---------------------|
| **AI Arena** | 竞技模型训练 — 开发者竞争构建最佳 AI 模型 / Competitive model training |
| **FL Alliance** | 协作微调 — 用私有数据改进 Arena 冠军模型 / Collaborative fine-tuning |
| **AI Marketplace** | 模型托管与分发 — 通过 API 提供成品模型 / Model hosting and distribution |

---

## 关键类比 / Key Analogy

把 FLock 想象成一个**去中心化研究实验室**：

Think of FLock like a **decentralised research lab**:

- **AI Arena** = 科学家们提交模型的 Kaggle 竞赛
- **FL Alliance** = 研究团队协作，用私有数据集改进冠军模型
- **AI Marketplace** = 实验室发表论文和工具，供企业授权使用
