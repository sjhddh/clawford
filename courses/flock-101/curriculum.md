# FLock 101 — 课程体系 / Curriculum

## 课程概述 / Course Overview

FLock 101 是 FLock.io 的基础课程——一个结合区块链与联邦学习的去中心化 AI 训练平台。学生通过实践学习：阅读文档、完成真实任务、在平台上构建应用。

FLock 101 is a foundational course on FLock.io — a decentralised AI training platform that combines federated learning with blockchain technology. Students learn by doing: reading docs, running real tasks, and building on the platform.

## 学习目标 / Learning Objectives

完成本课程后，学生将能够：

By the end of this course, students will be able to:

- **Explain / 解释** 什么是联邦学习及其重要性
- **Describe / 描述** FLock.io 三大核心产品架构（AI Arena、FL Alliance、FOMO）
- **Identify / 识别** AI Arena 中的四种参与角色（Task Creator、Training Node、Validator、Delegator）
- **Understand / 理解** FLOCK 代币如何激励生态参与
- **Use / 使用** FLock API 平台构建 AI 应用
- **Evaluate / 评估** FLock 是否适合特定应用场景

## 考核 / Assessment

- **评分标准 / Grading rubric:** 见 `exam.md` 和 `rubric.md`

## 教学风格 / Teaching Style

- 交谈式，而非照本宣科 / Be conversational, not lecture-y
- 使用类比和真实案例 / Use analogies and real examples
- 遇到难题时，承认不确定，而非不懂装懂 / When a student asks a hard question, admit uncertainty rather than bluffing
- 连接 FLock 概念与现实用例（医疗、金融、治理）/ Connect FLock concepts to real-world use cases
